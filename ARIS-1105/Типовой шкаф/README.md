Типовой шкаф
============

Архивы конфигураций для типовых шкафов ВЧ и ОПТИКА.

> !!! При установке на новое устройство необходимо снять галочки "Система", "Состав крейта" и "Настройки модуля". Остальные должны быть включены. !!!
